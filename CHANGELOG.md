# Changelog

All notable changes to the MojoDocs project will be documented in this file.

## [v1.4.2] - 2026-02-08
### Added
- **Code Protection Suite**: Integrated `webpack-obfuscator` for production binary protection.
- **Forensic Engine v2**: Enhanced Deepfake detection with structural noise analysis.
- **Multi-Threaded PDF Core**: New WASM-backed compression engine for 4x faster processing.
- **Proprietary Licensing**: Switched to strict MojoDocs IP protection model.

### Changed
- Improved UI responsiveness for PDF Compressor with "Smooth Camera" background.
- Optimized worker orchestration to prevent memory leaks in large image processing.

## [v1.3.8] - 2026-02-04
### Added
- **PDF Merger PRO**: High-fidelity merging with password-protected PDF support.
- **Mobile Optimized ToolView**: Unified footer and config sheets for better UX on small screens.
- **Passport Maker Elite**: Added A4 sheet generation for bulk professional printing.

### Fixed
- Resolved URI malformation issues in heavily obfuscated builds.
- Fixed canvas memory pressure during high-res background removal.

## [v1.2.5] - 2026-01-20
### Added
- **Background Remover v1.4**: Migrated to latest @imgly engine for 99% accuracy.
- **Local-First Analytics**: Privacy-safe event tracking without cloud storage.
- **Feature Prioritization**: Added smart sorting for tools based on user utility.

## [v1.1.0] - 2026-01-05
### Added
- **Image Editor Pro**: Saturation, Warmth, and Vignette controls with subject-blur logic.
- **WASM Support**: Initial integration of WebAssembly for image processing.
- **SponsorSnack System**: Interactive Indian snack visual integration for community support.

## [v1.0.0] - 2025-12-15
### Added
- **MojoDocs Alpha Launch**: Basic PDF and Image tools.
- **Privacy Engine**: Core local-only data processing architecture.
- **AEO/GEO Meta Engine**: Integrated SEO automation for better search visibility.
